# Panini Tracker

Web εφαρμογή για διαχείριση συλλογών αυτοκόλλητων Panini.

Λειτουργεί σε:

- Windows
- Android
- iPhone
- Tablet

Δεν απαιτεί εγκατάσταση ή δημιουργία λογαριασμού.

## Άνοιγμα εφαρμογής

Ανοίξτε:

https://konstadinosmarselis.github.io/panini-tracker/

## Εγκατάσταση ως εφαρμογή (PWA)

### Android

1. Ανοίξτε την εφαρμογή στον Chrome.
2. Πατήστε το μενού του browser.
3. Επιλέξτε «Προσθήκη στην αρχική οθόνη».

### iPhone / iPad

1. Ανοίξτε την εφαρμογή στο Safari.
2. Πατήστε Share.
3. Επιλέξτε «Προσθήκη στην αρχική οθόνη».

### Windows

Ανοίξτε την εφαρμογή σε Chrome ή Edge και πατήστε το εικονίδιο εγκατάστασης στη γραμμή διευθύνσεων.

## Πρώτη χρήση

1. Κατεβάστε το αρχείο λίστας άλμπουμ από τον φάκελο `docs`.
2. Επιλέξτε «Εισαγωγή λίστας άλμπουμ».
3. Φορτώστε το αρχείο.

## Δυνατότητες

- Εισαγωγή λίστας άλμπουμ
- Καταχώριση αυτοκόλλητων
- Ομαδική εισαγωγή
- Διαχείριση διπλών
- Αυτόματος έλεγχος διπλότυπων
- Αναζήτηση κωδικών
- Φίλτρο ανά ομάδα
- Ταξινόμηση ανά σελίδα ή κωδικό
- Εμφάνιση ελλείψεων
- Στατιστικά ανά ομάδα
- Εξαγωγή λίστας ελλείψεων
- Εξαγωγή και εισαγωγή backup JSON

## Αποθήκευση δεδομένων

Τα δεδομένα αποθηκεύονται τοπικά στον browser κάθε χρήστη.

Κάθε χρήστης έχει τη δική του συλλογή.

Δεν υπάρχει κοινός συγχρονισμός μεταξύ χρηστών.

Συνιστάται τακτική εξαγωγή backup JSON.
